# Kitano Life Finder

키타노역 입주 직후 필요한 생활필수품을 한 화면에서 정리하고, 라쿠텐·Amazon JP·Yahoo!쇼핑·메루카리·니토리·IKEA·무인양품·빅카메라 검색과 지도로 바로 이동하는 정적 사이트입니다.

## 포함 내용

- `체크리스트` / `지도` 탭 구조
- 도착 당일, 첫 주, 재택/업무, 출퇴근 대비 필터 + 빠른 검색
- D-day 카운트다운과 진행률, 우선순위 자동 정렬 체크리스트
- 검색 옵션 패널: 표시할 쇼핑몰, 예산 상한, 정렬, 送料無料 → 모든 검색 링크에 즉시 반영
- 가구류 사이즈 칩(매트리스·침대 기본 퀸) → 검색 키워드 반영
- 카테고리별 쇼핑몰 노출(가전=빅카메라, 가구=IKEA/니토리/무인양품 등)
- 키 없는 Google Maps 임베드 지도 + 매장/카테고리/기준 위치 연동
- 체크·검색 옵션·사이즈 상태 브라우저 저장, 라이트/다크 모드
- Vercel 정적 배포용 구성

## 프로젝트 의도

나중에 수정할 때는 [PROJECT_INTENT.md](./PROJECT_INTENT.md)를 먼저 읽으면 됩니다. 사이트의 목적, 사용 시나리오, 유지보수 기준을 정리해 두었습니다.
