# Kitano Life Finder

키타노역 입주 직후 필요한 생활필수품을 한 화면에서 정리하고, 라쿠텐/Amazon JP/니토리/빅카메라/Google Maps 검색으로 바로 이동하는 정적 사이트입니다.

## 포함 내용

- 도착 당일, 첫 주, 재택/업무, 출퇴근 대비 필터
- 매트리스, 침대, 책상, 의자, 세탁기, 조리도구 등 초기 생활품 체크리스트
- 키타노역/하치오지역 기준 지도 검색 전환
- 체크 상태 브라우저 저장
- Vercel 정적 배포용 구성

## 프로젝트 의도

나중에 수정할 때는 [PROJECT_INTENT.md](./PROJECT_INTENT.md)를 먼저 읽으면 됩니다. 사이트의 목적, 사용 시나리오, 유지보수 기준을 정리해 두었습니다.

## 로컬 실행

브라우저에서 `index.html`을 열면 됩니다. 별도 서버나 API 키가 필요 없습니다.

## 배포

```powershell
vercel --prod
```
